== Description ==

AwesomeOne Pro theme is responsive corporate WordPress theme which can useful for multipurpose industries. This theme is responsive and compatible with latest version of WordPress. This theme can also be useful for business, blogging, photography, magazine, religious, portfolio, counselling, restro etc. This theme has multiple features like color changing option, font typography, slider settings, multiple section and many more. This theme is very to use for non coders. This theme has created with the HTML5 and CSS3, AwesomeOne theme is cross borwser compatible with iphone and ipad and runs very smoothly. this theme is comaptible with plugins like woocommerce, contact form 7, jetpack etc. This is translation ready theme it can be translated in any language.
