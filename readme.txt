A starter theme called altitude-lite.

== Description ==
Altitude Lite is a Gutenberg compatible free WordPress theme. This theme is perfect for lawyers, attorneys, and related business owners.

== Installation ==

1. In your admin panel, go to Appearance > Themes and click the Add New button.
2. Click Upload Theme and Choose File, then select the theme's .zip file. Click Install Now.
3. Click Activate to use your new theme right away.

How to install the demo content.
1. Please go to Appearace-->Altitude lite option and click import demo content button.
2. If the Elementer plugin is not installed, please  install it and then start importing the demo content.

== Frequently Asked Questions ==

= Does this theme support any plugins? =
* Recommended plugins are Slidedeck, WP Legal Pages and Elementor

altitude-lite includes support for Infinite Scroll in Jetpack.

How to get the demo xml file:
Extract the downloaded free theme zip file.  You will find the demo xml file here : altitude-lite-->inc-->altitude-demo.xml

How to import demo xml file for a free Altitude (Altitude-lite) theme:
1. Firstly, download, install and activate free Elementor and WordPress Importer plugin.
2. Now go to Elementor—>setting and check the “Disable Default Fonts” and “Disable Default Colors” option. Save the changes. 
3. Go to tools-->import-->Wordpress (last option) and select run importer option.
4. It will ask to upload the xml file. Upload your xml file and select "upload file and import option"
5. It will take some time to upload. Once finished, move to pages-->All pages and view the Altitude Home page.
