== Description ==

Aerobics is a child theme of SwipeWp which is awesome free WordPress Business Theme comes up with plenty of powerful features that allow you to create a business website in any niche. It is integrated with WooCommerce and other popular plugins that offers full flexibility to use all its features. It has a promising design and exemplary features like Testimonials, Customization option, Optimized for SEO, Amazing support and documentation that makes no trouble while using the theme. Demo: http://demo.mysterythemes.com/swipe-demos/

== Installation ==

1. In your admin panel, go to Appearance > Themes and click the Add New button.
2. Click Upload Theme and Choose File, then select the theme's .zip file. Click Install Now.
3. Click Activate to use your new theme right away.
