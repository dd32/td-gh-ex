SunRain Responsive Corporate and Personal Theme

== Description ==

SunRain is CSS3 Powered and WordPress Latest Version Ready Responsive Theme.  SunRain has Multi Layer Slider where two Slides can show many Items which can be controlled from the Theme Options Panel.  Super Elegant and Professional Sunrain Theme will be adjusted automatically with all smart devices if the Responsive Layout be Selected.  Full Wide Template can be used for showing showcase or other pages. Other Templates will be useful for Page Builders. SunRain is Page Builder like, Elementor and E-Commerce ready by WooCommerce. You can Visit the SunRain Demo at http://demo.d5creation.com/wp/themes/sunrain

== Frequently Asked Questions ==

= What to do after Theme Installation ? =
After installing the Theme Please go to Appearance > Customize > SunRain Options. You can set the Slides, Featured Boxes Section, Headings, Social Links etc. there. 

= Why is SunRain ? =
SunRain is for Personal, Corporate, Community, Blog, Education, Entertainment etc. Websites

= Can it be used for Personal or Company Websites? =
You can use for Personal to Corporate Sites and Even for Smart Blogs

= Is it only for Blogs ? =
No, It is for Blog, Company, Personal, Education, Portfolio, Entertainment etc. Websites.
