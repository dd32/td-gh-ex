INSTALLATION

1. Unzip artsavius.zip

2. Upload artsavius folder to /wp-content/themes

3. Activate Artsavius Blog Theme on the Theme admin page

4. Upload your own avatars in /wp-content/themes/artsavius/logo

5. Go to Artsavius Blog Options page to configure the theme

6. Enjoy
