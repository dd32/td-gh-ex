ABOUT THEME
Big Blue is a classic blogging theme, with a light on dark color scheme. It offers an optional home page slider which can be managed in the customizer. 
The customizer also provides simple control of the main theme colors, header image, background image, social media links as well as the ability to add your own custom CSS.

For free themes support, please contact us http://hostmarks.com/contact/
