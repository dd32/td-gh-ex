== Description ==

Our Masonry theme is based on the popular and efficient grid library, suitable for creatives, photographers or bloggers. Although its nice interface and features make it an ideal WordPress theme for almost anything, Masonry has been optimized for all devices thanks to our latest responsive, mobile-first design techniques.

== Installation ==

= From within WordPress =
1. Visit "Appearance > Themes > Add New"
2. Search for "Masonry"
3. Install and activate

== Social Menu ==

To add social icons in your header you need to go to Dashboard => Appearance => Menus and to create new menu.
From Theme locations select Social and save the menu.
To add your social profiles select Links and in URL add your Social profile URL.
E.g for Facebook - http://facebook.com/yourprofile, for Twitter - http://twitter.com/yourprofile
