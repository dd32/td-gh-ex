== Description ==
Arrival Store is a complete eCommerce theme. The theme is extended in ecommerce stores in mind, so you can use it for your online stores and sell your product in style. The theme is fully built for page builders you can use any page builder to customize it according to your need.


== Install Steps: ==

1. Activate the theme
2. Go to the Customize page
3. Setup theme options
