== Description ==

Attimo is a creative and clean minimal child theme of Avventura Lite, perfect to build your personal, travel, beauty, lifestyle, news, photography website. Attimo offers a sticky post grid on homepage, an optional banner area on header and a new layout for the navigation menu, social buttons and Google Fonts.

Developed by ThemeinProgress, https://www.themeinprogress.com
