== Description ==
Clean and Simple WordPress Theme designed for blogs and online publications. With easy to use options The Box lets you focus on your awesome writing.


== Frequently Asked Questions ==

= How to manually install a WordPress theme? =
1. After downloading the zip file, go to Appearance > Themes and click on the *Install Themes* tab.
2. Click on the *Upload* link.
3. Upload the zip file that you downloaded and click *Install Now*.
4. Click *Activate* to use the theme you just installed.

= How do I Setup my theme? =
Please refer to our theme documentation:
https://www.designlabthemes.com/the-box-documentation/

= How can I get Support? =
Please visit https://wordpress.org/support/theme/the-box if you have any questions about the theme.
