== Description ==

Bicubic Wordpress theme is simply solution to customize your website. It`s clean, simply and very powerful tool for organizing your blog or others categories of websites..
You may change some design elements from Admin UI, like: background of page, header image.

== Installation ==

Manual installation:

1. Upload the `Bicubic` folder to the `/wp-content/themes/` directory
2. Activate the theme through the 'Themes' menu in WordPress

Installation using "Add New Theme":

1. From your Admin UI (Dashboard), use the menu to select Themes -> Add New
2. Search for 'Bicubic'
3. Click the 'Install' button to open the theme's repository listing
4. Click the 'Install' button
5. Activate the theme through the 'Themes' menu in WordPress

== Theme Notes ==

= Featured Images =

For featured images of posts or pages we recommend using the proportions 540x280px (width/height).

= Custom background =

Theme support custom background-color. Defaul color is #eeeeee

= Custom Menus =

Theme supports custom menu in one location "primary" in header.

= Widgets =

The Theme includes a standart widgets.
By default in sidebar will display widgets: recent posts, recent comments, archive and categories.
You can change the set of widgets on your own.

= Translation-ready =

Theme is translation-ready. Current available translations:
* Russian (ru_RU)

== Frequently Asked Questions ==

= How to use custom-bachground? =

You can change color of background very simply throught Admin UI (Dashboard): use menu to select Appearance -> Customise -> Colors -> Background color

= How can I create a custom menu? =

You need to go to Dashboard and use menu to select Appearance -> Menus. You must be at 'Edit Menu' tab, find the button "or create new menu" and click it. Then just drag and drop needed pages from the list of pages to menu. Don't forget save your new menu after all.

= How to set custom menu to primary menu section? =

You need to go to Dashboard and use menu to select Appearance -> Menus. Tnen go to 'Manage locations' tab and select menu that you want to set.

= How to add widgets to sidebar? =

You just need go to Dashboard and use menu to select Appearance -> Menus. Then drag and drop needed widgets to sidebar area. That's all.

= How to change language from english to russian? =

In main catalog of your WordPress site find file wp-config.php, open it and change define (‘WPLANG’); to define (‘WPLANG’, ‘ru_RU’
