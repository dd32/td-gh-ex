== Description ==
Asteroid is a simple, clean and responsive WordPress theme. It incorporates various widget hooks that make management of banners and ads a lot easier, especially for ad-supported sites. 
The theme's development focuses on customizations and resource efficiency, helping to minimize server load. All for free.
