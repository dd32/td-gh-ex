Installation Instructions:
1. Unzip this file into your mywebsite.com/wp-content/themes/ directory, should look like mywebsite.com/wp-content/themes/adventurejournal/
2. Go into your Wordpress Admin, navigate to 'Appearance > Themes'
3. Find the Adventure Journal listing on this page and click 'Activate'
