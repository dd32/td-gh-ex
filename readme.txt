ABOUT THEME
=========== 

Responsive WordPress Theme.This theme made with Responsive Framework Skeleton v2.0.4. Used some css3 effects.

This theme is compatible with Wordpress Version 3.4 and above and it supports the new theme customization API (https://codex.wordpress.org/Theme_Customization_API).


Supported browsers: Firefox, Opera, Chrome, Safari and IE9+ (Some css3 styles like shadows, rounder corners are not supported by IE8 and below).
