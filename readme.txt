== Description ==

Bayleaf is a lightweight and mobile first WordPress theme that is designed for cooking or food blog. It provides many tools and widgets to create an awesome food blog website. Bayleaf is written with clean code and the most current HTML5 conventions. Bayleaf is also compatible with Gutenberg and new WordPress block editor.

== Installation ==

1. In your admin panel, go to Appearance > Themes and click the Add New button.
2. Click Upload and Choose File, then select the theme's .zip file. Click Install Now.
3. Click Activate to use your new theme right away.
4. Navigate to Appearance > Customize in your admin panel and customize to taste.
