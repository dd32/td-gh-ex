== Arise ==
Arise is a Responsive Multi-Purpose Business WordPress Theme, Designed and Developed by Theme Freesia. This theme supports popular plugins like Breadcrumb NavXT, WP-PageNavi, Contact Form 7, Jetpack by WordPress.com, Polylang, WPML Compatible, Newsletter, bbPress and wooCommerce. It is also translation ready and currently translated in French language. It is also easily customizable with lots of theme options using Customizer. It consists contact us template, gallery template and many widgets and sidebar. Get free support at http://themefreesia.com/support-forum/ and View demo site at http://demo.themefreesia.com/arise

== Installation ==
1. Primary: Login to your wp-admin area and go to Dashboard -> Appearance -> Themes. Select Install tab and click on Upload link. Select theme .zip and click on Install now button. If you have any errors, use can also use an alternate method.

2. Alternate: Unzip the template file (arise.zip) that you have downloaded. Via FTP, upload the whole folder (arise) to your server and place it inside the http://yourdomain.com/wp-content/themes/folder(arise). Do not change directory name. The template files should thus be here now: http://yourdomain.com/wp-content/themes/arise/index.php (for example).

3. Log into your WP admin panel and click on the Appearance -> Theme tab. Now click on the Arise theme to activate it.

4. Want to enjoy Arise feature. Go to  Dashboard -> Appearance -> Customize and play with Arise Options

5. The theme is offered free of charge. If someone asked money for it, someone just tricked you.
