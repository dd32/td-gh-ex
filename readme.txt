## About Aladdin

Description: Aladdin helps you to build a really magical one page website. The main feature of Aladdin is parallax sections with custom parallax speed. Any number of sections can be featured on the front page. Each section can contain page, slider, gallery, portfolio, products and any other WordPress widget. Aladdin has lot's of other options in the theme customizer: layouts for right, left, two columns, 4 blog layouts, all google fonts, social media icons, a number of widgets for the portfolio. Theme includes their own sidebars for Blog, Home, Portfolio, Shop and per page widgets. Aladdin has responsive design and can be used across a wide range of devices (from mobile phones to desktop computers). Supports popular plugins: portfolio from Jetpack and WooCommerce. Theme demo can be found at http://wpblogs.info/demo/aladdin/.

## How to install

	1. Unzip and then copy the aladdin directory to /wp-content/themes/ in your WordPress installation. 
	2. Activate from admin menu. 

## How to use

* Theme options placed at "Appearance >> Customize"
* Widgets can be found at "Appearance >> Widgets"
