== Description ==
Bayn Lite is a clean WordPress portfolio theme which is made for design and architecture businesses. Bayn Lite has a full-width slider and focuses on showing the photos on the website, so you could also use Bayn Lite as a photography theme.

Built with great care, Bayn Lite is compatible with all modern browsers, responsive on all screens, and optimized for speed.

* Full-width slider
* Featured services
* Custom logo
* Jetpack supported

For more information about Bayn Lite please go to https://gretathemes.com/wordpress-themes/bayn/.
