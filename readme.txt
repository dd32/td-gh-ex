== Theme Description ==

Customizable is a WordPress responsive blog theme based on Twitter bootstrap framework which can be used for personal or corporate websites. Customizable is a free WordPress theme with premium functionality and design. 

== Features ==

Responsive design(Bootstrap), Theme Options, Custom Widgets, Custom Menu, supports social media, Right-Sidebar, Full-width.

== Installation ==

1. Primary:

 = Login to your wp-admin account and go to Appearance -> Themes.

 = Select "Install" tab and click on the "Upload" link.

 = Select "theme.zip" and click on "Install Now" button.

 = In case of errors, use an alternate method.



2. Alternate:

 = Unzip the template file (customizable.zip) that you have downloaded.

 = Upload the entire folder (customizable) to your server via FTP and place it in the /wp-content/themes/folder.

 = Do not change the directory name.

 = The template files should be there now: /wp-content/themes/customizable/index.php (example).

3. Log into your WP admin panel and click on "Appearance". Go to "Themes" tab.

4. Now click on "customizable" to activate the theme.

5. Complete all of the required inputs in the theme Options page (in the WP admin panel) and click "Save".



6. Basic Settings:

 = Login to your wp-admin area and go to Appearance -> Theme Options.

 = Upload the logo, favicon and set Home page heading Text and Copyright Text.
