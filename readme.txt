== Description ==

fCorpo is a Child Theme of fKidd Theme. It's suitable for Corporate, Business,
Company, Startup website or blog. The theme is Fully Responsive, Built-in Slider,
Multi-Level Drop-Down Custom Menu, Admin Panel Options, W3C Markup Validated,
Search Engine Optimized, Multiple Browsers support (IE9, IE10, IE11, Chrome,
Firefox, Safari, and Opera), Page navigation, Threaded Comments, FontAwesome Icons,
Custom Post Formats, Translation Ready, and much more.

== Installation ==

Manual installation:

1. Upload the fCorpo and fKidd folders to the /wp-content/themes/ directory

Installation using 'Add New Theme'

1. From your Admin UI (Dashboard), use the menu to select Themes -> Add New
2. Search for 'fCorpo'
3. Click the 'Install' button to open the theme's repository listing
4. Click the 'Install' button

Activiation and Use

1. Activate the Theme through the 'Themes' menu in WordPress
2. See Appearance -> Theme Options to change theme specific options
