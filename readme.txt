Share your adventures with the world! Adventure Journal allows you to select header art, backgrounds, custom stylesheet, and a one, two or three column layout from the comfort of your Dashboard - no coding required!

== Installation ==

= Via WordPress Admin =
1. From your sites admin, go to Themes > Install Themes
1. In the search box, type 'Adventure Journal' and press enter
1. Locate the entry for 'Adventure Journal' (there should be only one) and click the 'Install' link
1. When installation is finished, click the 'Activate' link

= Manual Install =
1. Unzip this file into your mywebsite.com/wp-content/themes/ directory, should look like mywebsite.com/wp-content/themes/adventurejournal/
2. Go into your Wordpress Admin, navigate to 'Appearance > Themes'
3. Find the Adventure Journal listing on this page and click 'Activate'
