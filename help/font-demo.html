Weaver Xtreme Font Examples

Web Fonts

The following Web fonts are supported by most modern desktop browsers. Ideally, you should see examples of the named font. If your browser doesn't support a specific Web font, it will use a fallback font. Many mobile browsers, especially Android based browsers, will not show many of these examples, and you'll see many of the look the same. If you use the Google Fonts supported by Weaver Xtreme Plus for your site, they will be correctly displayed on essentially all modern browsers.

Pre-Defined Google Fonts (Weaver Xtreme Plus)

Note: The following 36 Google Font Examples font sets, included with the Weaver Xtreme Plus pre-defined Google Fonts, are all dynamically loaded. That should give you a good example of just how fast (or slowly) they load. In most browsers, you should see a slight delay in the display of the following examples as the fonts are loaded. You can click the browser page reload to see this happen. Some browsers will cache the fonts, so you may see no delay after the page has been initially loaded. Note that 36 is a lot of fonts, and using just 4 or 5 loadable fonts will unlikely have much of a performance hit on your site.
