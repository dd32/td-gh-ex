/*========= Installation =========*/

1. Upload the 'Avis-lite` folder to the `/wp-content/themes/` directory
Activation and Use
1. Activate the Theme through the 'Themes' menu in WordPress
2. See Appearance -> Avis Options to change theme options.
3. Create a page and set as Front Page.
4. See Settings -> Reading -> Choose a Page for the Front page and Page for Posts page from drop-down.


/*========= Theme Features and Documentation =========*/

== WP Version ==

It's requires at least version 4.1 to work properly. Please make sure your WP version is at least 3.8.


== Theme Setup ==

Go To Appearance -> Avis Options For Theme Options.

*General Settings:*
1)Primary Theme Color
2)Secondry Theme Color
3)Change Favicon

*Header Settings:*
1) Change Logo.
2) Logo Width.
3) Logo Height.

*Header Image Setting :*
1) Static Background Image.

*Home Featured Section
1) Enable/Disable Featured box section.
2) Featured box section heading.
3) Featured box section description.
 
Set featured boxes from widget(go to Appearance >> Widgets and use Home Featured Sidebar)

*Breadcrumb Settings:*
1) Breadcrumb Holder Background Image.
2) Breadcrumb Background Image Position.
3) Breadcrumb Background Image Size.

*Blog Settings :*
1)  Enter Blog Page Title.

*Footer :*
1) Enter Your Copyright Text (You can use HTML for links.)
2) Footer Social Links.
