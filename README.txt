How to install: 
1. Unpack this archive in your wp-content/themes/ directory.
2. Go to Admin WordPress and select presentation.
3. Select 3COLOURS theme

Plugins need:

1. Ultimate Tag Warrior available at http://www.neato.co.nz/ultimate-tag-warrior/
2. Recent Comments available at http://blog.jodies.de/archiv/2004/11/13/recent-comments/
3. Fickr RSS available at http://eightface.com/wordpress/flickrrss/

Installing the plugins:
1. Unpack each plugin archive in the wp-content/plugins/ directory.
2. Go to your WordPress dashboard then select plugins.
3. Active the Recent Comments and Ultimate Tag Warrior. You don't need to change or add any codes because it's already configured for the plugins.
4. You'll need to insert the php code <?php get_flickrRSS(); ?> somewhere in your templates
5. You have the i box installed within this theme, it's similar to the lightbox plugin, but lightweighted,
for use add this code to the header:
<script type="text/javascript" src="<?php bloginfo('stylesheet_directory'); ?>/js/ibox.js" ></script><link rel="stylesheet" href="<?php bloginfo('stylesheet_directory'); ?>/css/ibox.css" type="text/css" media="screen" />
6. This code is for use in posts, pages or sidebars:
<a rel="ibox" href="http://adress of original pic" title="mane"><img class="alignright or alignleft or centered" src="http://adress of thumbnail pic" alt="name" /></a>
