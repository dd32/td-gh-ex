== Description ==

Content is a responsive HTML5 WordPress theme; 
it is widget ready with six widgets area to be customized or used in default mode: every widgets are placed in the footer; 
Content WordPress theme is an elegant theme focused on readability.
The font used is PT Sans ( http://www.google.com/fonts/specimen/PT+Sans ).

#### Notes
* Content theme is a free WordPress theme, always remember to do a backup of your whole site before install and try it: 
do a backup of both filesystem and database.

A note about the menu:
Content default menu will does not show sub-levels menu items.

== Installation ==

01. Download Content theme ( http://ramacciotti.altervista.org/archives/articles/content-wordpress-theme ),
02. Upload via FTP the `Content` folder to your `/wp-content/themes/` directory,
03. Activate through the 'Appearence -> Themes' menu in WordPress,
04. Optional: customize the theme options like Widgets and Menu setup.

Remember to remove manually the Search Widget from the main widgets area, if you have it: it is already in the header of the site: no need to have the search form twice.
