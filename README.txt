== Description ==

AdStyle is designed for adsense placement, has  modern scroll work and broken boundries for added flair. 
Your adsense code needs to be added in the wp-admin panel. 


Recommended Colors:
	Border = #FDF4E2
	Title = #DA944A
	Background = #FDF4E2
	Text = #54340D
	URL = #AEC2E1

Thank You, Gordon French

== Installation ==

1. Upload `adStyle` to the `/wp-content/themes/` directory
1. Activate the theme through the 'Themes' menu in WordPress
1. Place your adsense code in the textarea under adStyle
