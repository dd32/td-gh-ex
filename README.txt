Bahama WordPress Theme
http://www.studiopress.com/themes/bahama

INSTALL: 
1. Upload the theme folder via FTP to your wp-content/themes/ directory.
2. Go to your WordPress dashboard and select Appearance.
3. Select Bahama WordPress theme.
4. Inside your WordPress dashboard, go to Appearance > Bahama Theme Options and configure them to your liking.

SIDEBAR WIDGETS:
Sidebar - This is the main sidebar if you are using the Sidebar/Content, Content/Sidebar or Sidebar/Content/Sidebar Blog Layout option.
Sidebar Alt - This is the left-hand sidebar if you are using the Sidebar/Content/Sidebar Blog Layout option.
