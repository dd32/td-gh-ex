Description: 2 Column, widget-ready fixed width theme sporting a basic elegance.  Featuring an author info page, gravatars and author box along with Custom Archives, Sitemap and full width No Sidebars page templates.  Tested in Firefox, Opera, Safari, IE6 and IE7.  Compatible up to Wordpress 2.7.

HORIZONTAL NAVIGATION:
This theme comes with a horizontal menu bar in the header.  For those that prefer a separate menu bar under the header can go into header.php and follow the instructions there.

INSTALLATION: 
1. Unzip and upload the directory to your wp-content/themes/directory.
2. Go to Admin >> Design and select the theme.
3. Go to your site to see the theme.
