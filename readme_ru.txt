== Description ==

Bicubic Wordpress theme is simply solution to customize your website. It`s clean, simply and very powerful tool for organizing your blog or others categories of websites..
You may change some design elements from Admin UI, like: background of page, header image.

== Installation ==

Manual installation:

1. Upload the `Bicubic` folder to the `/wp-content/themes/` directory
2. Activate the theme through the 'Themes' menu in WordPress.

Installation using "Add New Theme":

1. From your Admin UI (Dashboard), use the menu to select Themes -> Add New
2. Search for 'Bicubic'
3. Click the 'Install' button to open the theme's repository listing
4. Click the 'Install' button
5. Activate the theme through the 'Themes' menu in WordPress

== Theme Notes ==

= Featured Images =

For featured images of posts or pages we recommend using the proportions 540x280px (width/height).

Для прикрепленных изображения постов и страниц мы рекоммендуем использование размеров 540 на 280 пикселов ( ширина/высота )

= Custom background =

Theme support custom background-color. Defaul color is #eeeeee

Тема поддерживает пользовательский цвет фона. Цвет по умолчанию #eeeeee


= Custom Menus =

Theme supports custom menu in one location "Primary menu" in header.

Тема поддерживает пользовательское меню в одном месте "Главное меню" в шапке.

= Widgets =

The Theme includes a standart widgets.
By default in sidebar will be displayed these widgets: recent posts, recent comments, archive and categories.
You can change the set of widgets on your own.

Тема включает в себя стандартные виджеты.
По умолчанию в боковой части будут отображены такие виджеты: недавние посты, недавние комментарии, архивы и категории.
Вы можете изменить набор виджетов по вашему выбору.

== Frequently Asked Questions ==

= How to use custom-bachground? =
= Как использовать пользовательский фон?

You can change color of background very simply throught Admin UI (Dashboard): use menu to select Appearance -> Customise -> Colors -> Background color
Вы можете изменить цвет фона очень просто с помощью Admin UI (Dashboard): используйте меню, чтобы выбрать Appearance -> Customise -> Colors -> Background color

= How can I create a custom menu? =
= Как я могу создать пользовательское меню?=

You must go to Dashboard and use menu to select Appearance -> Menus. You must be at 'Edit Menu' tab, find the button "or create new menu" and click it. Then just drag and drop needed pages from the list of pages to menu. Don't forget save your new menu after all.
Вы должны зайти в Dashboard и использовать меню для выбора Appearance -> Menus. Вы должны оказаться на вкладке 'Edit Menu', найдите кнопку "or create new menu" и нажмите на нее. Потом просто перетаскивайте необходимые страницы из списка страниц в меню. Не забудьте сохранить ваше новое меню после всего.

= How to set custom menu to primary menu location? =
= Как назначить пользовательское менб в главную секцию?
You must go to Dashboard and use menu to select Appearance -> Menus. Tnen go to 'Manage locations' tab and select menu that you want to set.
Вы должны зайти в Dashboard и использовать меню для выбора Appearance -> Menus. Потом переходите к вкладке 'Manage locations' и выбирайте меню, которое вы хотите назначить.

= How to add widgets to sidebar? =
= Как добавить виджеты в боковую панель?=

You just need go to Dashboard and use menu to select Appearance -> Widgets. Then drag and drop needed widgets to sidebar area. That's all.
Вам прсото нужно перейти в Dashboard и использовать меню для выбора Appearance -> Widgets. Потом перетаскивайте необходимые виджеты в область боковой панели. Это все.

= How to change language from english to russian? =
= Как поменять язык с английского на русский?+

In main catalog of your WordPress site find file wp-config.php, open it and change define (‘WPLANG’); to define (‘WPLANG’, ‘ru_RU’)
Найдите файл wp-config.php в главном каталоге вашего WordPress сайта, откройте его и измените define (‘WPLANG’); на define (‘WPLANG’, ‘ru_RU’);
